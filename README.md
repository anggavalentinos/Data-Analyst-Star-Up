# Data-Analyst-Star-Up

Bisnis Problem
Latar Belakang: Sebuah perusahaan distribusi yang terlibat dalam pengelolaan produk ke berbagai wilayah melalui saluran distribusi tertentu ingin meningkatkan efisiensi dan profitabilitas operasinya. Perusahaan menghadapi tantangan dalam mengidentifikasi kinerja distributor, produk, dan wilayah mana yang memberikan kontribusi paling besar terhadap penjualan serta area yang memerlukan perhatian lebih.
